Политика конфиденциальности

Дата публикации: 2026 г.

1. Общие положения

Настоящая политика обработки персональных данных составлена в соответствии с требованиями Федерального закона от 27.07.2006 № 152-ФЗ «О персональных данных» и определяет порядок обработки персональных данных и меры по обеспечению безопасности персональных данных, предпринимаемые Администрацией сайта Стройиндустрия (далее — Оператор).

2. Персональные данные, которые обрабатывает Оператор

  • Фамилия, имя, отчество.
  • Номер телефона.
  • Адрес электронной почты (при наличии).
  • Адрес объекта строительства (для замера).
  • Обезличенные данные о посетителях (в т.ч. файлы «cookie») с помощью сервисов интернет-статистики (Яндекс Метрика и др.).

3. Цели обработки персональных данных

  • Предоставление консультаций по услугам бетонных работ и расчет стоимости.
  • Установление обратной связи, включая направление уведомлений и запросов.
  • Заключение договора подряда на строительные работы.

4. Правовые основания обработки

Оператор обрабатывает персональные данные Пользователя только в случае их заполнения и/или отправки Пользователем самостоятельно через специальные формы на сайте или через мессенджеры (WhatsApp/Telegram).

5. Порядок сбора, хранения, передачи и других видов обработки

  • Безопасность персональных данных обеспечивается путем реализации правовых, организационных и технических мер.
  • Персональные данные никогда, ни при каких условиях не будут переданы третьим лицам, за исключением случаев, связанных с исполнением действующего законодательства.
  • Срок обработки персональных данных является неограниченным. Пользователь может в любой момент отозвать свое согласие, направив уведомление Оператору.

6. Заключительные положения

Пользователь может получить любые разъяснения по интересующим вопросам, касающимся обработки его персональных данных, обратившись к Оператору по телефону 8 (927) 934-44-48.
